## To Install this project

- Step 1: Composer Install
    `composer install`
- Step 2: Generate Key
    `php artisan key:generate`
- Step 3: Migrate
    `php artisan migrate`
- Step 4: NPM Install
    `npm install`
    `npm run build`
- Step 5: Create storage link
    `php artisan storage:link`
